🔐 Générateur de hash

Outil local pour générer le hash SHA-256 de votre mot de passe admin.
Ce hash sera stocké comme secret GitHub (ADMIN_PASSWORD_HASH).

Hash SHA-256 (hexadécimal)
Étapes suivantes :
1. Copiez ce hash.
2. Dans GitHub → Settings → Secrets → Actions, créez le secret ADMIN_PASSWORD_HASH avec cette valeur.
3. Supprimez l'historique de cet outil de votre navigateur si vous le souhaitez.
